## Per-leaf tests
For each leaf:
- HELP   — `<leaf> --help` exits 0 with Examples
- HAPPY  — realistic invocation with --json (or default for print/open) exits 0
- JSON   — happy-path output parses as valid JSON (skip when text-only: print, open)
- ERROR  — bad arg invocation exits non-zero (skip when leaf takes no required arg: list)

Operational commands skip ERROR (they don't take args).

Total: ~19 domain leaves × 3-4 + 4 operational × 1-2 ≈ 65 tests.
